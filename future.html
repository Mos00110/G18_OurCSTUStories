อนาคตที่พวกเราหวังไว้หลังจากเรียนจบคือ
การที่พวกเราได้ทำงานในสิ่งที่ตนเองถนัดหรือชื่นชอบ เพราะการที่ได้ทำงานที่ชอบจะทำให้งานที่ทำออกมามีประสิทธิภาพมากขึ้น และพวกเราก็หวังว่า จะได้เจอเพื่อนร่วมงานที่ดีในระหว่างการทำงาน และหวังว่าในอนาคตจะได้ทำธุรกิจที่อยากทำ


อนาคตที่เราแต่ละคนคาดหวังไว้

Name : Natthaphat
Plan : นอนอย่างอร่อย มีบริษัทส่วนตัวที่นอนและทำงานเมื่อไหร่ก็ได้ หวังว่าจะนอนและเล่นเกม ตื่นมาเล่นกับหมาแมว มีเงินไว้ซื้อของตามใจได้

Name : Wanwisa
Plan : อยากทำ Software Tester อยากทำงานในบริษัทไหนก็ได้ที่เราเข้าไปแล้วเรามีความสุขที่จะอยู่ตรงนั้นและบริษัทที่มั่นคง หวังว่าจะประสบความสำเร็จในหน้าที่การงานมั่นคงเลี้ยงดูครอบครัว ทำธุระกิจส่วนตัว

Name : Kuntanat
Plan : อยากทำ Full-Stack Developer อยากทำงานในบริษัทใหญ่ บริษัทที่มีเพื่อนร่วมงานช่วยเหลือและสามารถสื่อสารต่อกันได้ง่าย และมีความสุขที่ได้อยู่ในบริษัทนั้นๆ หวังว่าจะทำงานในบริษัทใหญ่ที่ตนเองชอบ ได้รับเงินเดือนที่มากและมั่นคง ดูแลคนที่รักและเลี้ยงแมวสัก2ตัว

Name : Kantaphat
Plan : อยากทำ game-dev อยากทำงานในบริษัทเกม หวังว่าจะได้เป็นโปรแกรมเมอร์ในบริษัทใหญ่ๆที่มั่นคง

Name : Charatrawi
Plan : อยากทำ Data Scientist อยากทำงานในบริษัทใหญ่ที่มีวัฒนธรรมแบบที่ไม่มีระบบอาวุโส สนใจในคุณภาพงานมากกว่าอายุและหวังว่าจะเจอเพื่อนร่วมงานที่ดี หวังว่าจะได้ทำงานที่ชอบแบบมีแพชชั่นในทุกๆวัน มีเงินเดือนที่ดี เลี้ยงหมา1ตัว และเกษียณตอนอายุ 60 ปี

Name : Nattakritta
Plan : อยากทำงานเกี่ยวกับสายที่เรียน บริษัทที่เพื่อนร่วมงานดี เราอยู่แล้วมีความสุข หวังว่าจะมีงานที่มั่นคง มีเงินพอที่จะไม่ต้องมากังวลกับเรื่องค่าใช้จ่ายต่างๆ สามารถบาลานซ์ระหว่างเวลางานกับเวลาพักผ่อนได้ แล้วก็อยากเลี้ยงแมวที่นิสัยอ้อนๆติดคนสัก1ตัว
